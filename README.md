# chatserver-nodejs

Cung cấp chát cá nhân, chát nhóm cho nội bộ doanh nghiệp.

cách sử dụng:
sau khi clone về thì mở terminal gõ: npm i
Tạo 1 file .env sau đó trong file thêm

PORT = port bạn tạo
JWT_SECRET = mã jsonwebtoken của bạn

sau đó chạy: npm run dev
